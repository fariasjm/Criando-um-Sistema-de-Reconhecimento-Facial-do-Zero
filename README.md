# Criando um Sistema de Reconhecimento Facial do Zero

## Objetivo
  A proposta deste projeto é implementar um sistema de detecção reconhecimento facial em tempo real usando Python.
  Utilizando o framework TensorFlow em conjuntos com as bibliotecas que o projetista julgue necessárias, de forma ilimitada.

O sistema deve ser capaz de detectar e reconhecer mais de uma face.
